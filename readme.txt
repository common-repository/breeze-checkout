== Installation ==

#### Automatic Installation

To download and install the Breeze Checkout plugin, make sure you follow the steps below:

- Navigate to the WordPress  Dashboard > Plugins.
- Click the Add New button and type in the Search Bar "Breeze Checkout".
- When the plugin appears, click on the Install button and then on  Activate.

#### Manual Installation

- Log into your website admin page, then go to Appearance > Plugins. You will see all the plugins currently installed in your dashboard listed there.
- To install a new WordPress plugin, simply click on the Add New tab.
- Click the Upload Plugin button in the WordPress Dashboard.
- Click the Choose File button, select the ZIP archive of the Breeze Checkout plugin from your local computer, and press the Install Now button.
- Click on the Activate button under the "Plugin installed successfully" message.

If you want to upload the plugin using FTP, [click here](https://themeisle.com/blog/what-is-ftp-wordpress/) to see a detailed explanation about it.

== Frequently Asked Questions ==

= Where are the settings of this plugin ? =
Visit "WooCommerce > Breeze".

= Can I use this checkout button on product display page ? =
Yes, Breeze checkout button works on product page, mini cart page (cart drawer), cart page and checkout page

= Do you support offers? = 
Yes, Breeze provides full support for Woocommerce offers and more. Your customers can apply both store offers and payment offers in one seamless interface.

= Do you support Refund ? =
Yes, Breeze provides full fledge end to end working refund flow, which enables creating refund request in Dashboard.

= Do you support Partial Payment ? =
Yes, Breeze provides partial payment with customization, like changing either amount or percentage wise.

= Do you support Prepaid Offer ? =
Yes, Breeze provides prepaid offer supports when customer purchase by doing payment via any prepaid instruments.
